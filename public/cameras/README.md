# Camera Image Sources

These real camera photos are sourced from Wikimedia Commons and are used as local dashboard assets.

- `nikon-fm2.jpg`: Wikimedia Commons, `File:Nikon FM2.jpg`, JorgeGT at en.wikipedia.
- `canon-ae-1.jpg`: Wikimedia Commons, `File:Canon AE-1.JPG`, Wasabi215, public domain.
- `canon-af35ml.jpg`: Wikimedia Commons, `File:Canon ML (3826136324).jpg`.
- `olympus-xa.jpg`: Wikimedia Commons, `File:OlympusXA.JPG`, Martintoy, CC BY-SA 3.0.
- `mamiya-645.jpg`: Wikimedia Commons, `File:Mamiya 645.jpg`, Lachy, CC BY 2.0.
- `pentax-k1000.jpg`: Wikimedia Commons, `File:Pentax K1000.jpg`, Martin Taylor, CC BY-SA 2.5.

The `*-cutout.png` files are local alpha PNG derivatives generated from these sources with a light-background keying pass.
The app falls back to generated SVG camera art for camera names that do not have a matching local cutout.
